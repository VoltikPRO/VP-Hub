Usage

How to use the Lightning Protection Plugin

Workflow

  1. Save your project and make sure the worksets are not locked by other users.
  2. Select the elements (Masts) that should be included in the calculation.
  3. Choose an appropriate Mesh type (radius) from the list. Note: If the family is missing in your project, it will be imported automatically.
  4. Run the calculation. It may take about 2–5 minutes to complete.

✅ Done. The corresponding Mesh elements will be generated between the mast tops.

Notes

  • If something looks wrong, use Current Calculation Result to identify potential issues.
  • If you encounter a bug, go to About and leave your feedback.

Enjoy using the plugin!
