How to install the LP Revit Plugin

Steps

  1. Download the latest version of the plugin from GitHub.
  2. Extract the .zip archive.
  3. Copy the .dll and .addin files into your Revit Addins folder:
    • %AppData%\Autodesk\Revit\Addins\[Revit Version]
  4. Start Revit – the plugin will appear in the ribbon.

Note

If updating the plugin, simply replace the old .dll and .addin files in the same folder.
