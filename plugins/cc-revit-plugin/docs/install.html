Встановлення VP Revit Plugin

Кроки встановлення

  1. Завантажте останню версію плагіна з GitHub Releases .
  2. Розпакуйте архів .zip.
  3. Скопіюйте файли .dll та .addin у відповідну папку Revit:
    • %AppData%\Autodesk\Revit\Addins\[версія Revit] (Windows)
  4. Запустіть Revit — плагін з’явиться у стрічці інтерфейсу.

Примітка

Якщо ви оновлюєте плагін — просто замініть старі .dll та .addin файла у тій же папці.
